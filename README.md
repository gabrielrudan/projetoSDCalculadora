# Sistemas Distribuídos e Redes De Comunicação - AT03
## Invocação Remota e Comunicação Indireta

Requisitos Calculadoras
1. Os clientes das calculadoras devem ter uma interface gráfica mínima (textual ou visual) 
2. Devem suportar operações individuais (e.g., 10 + 15) como expressões matemáticas com parênteses (e.g., (10 + 15) * 4). As expressões devem ter duas implementações (uma em que são decompostas nos clientes que invocam as operações básicas no servidor e outra onde é calculada inteiramente no servidor).
3. Você pode utilizar os projetos dos professores, mas deve comentar cada linha do código das classes de comunicação indicando o que a linha de comando faz. 
4. Podem escolher outras linguagens de programação
5. O cliente HTTP deve implementar uma política de RETRY (e.g., Wait Ts and Retry)
